# Human-Resource-Attrition
# Human Resource Attrition Analysis & Prediction

## Overview

Employee attrition is a major concern for organizations because it leads to increased recruitment costs, loss of organizational knowledge, and reduced productivity. This project explores employee attrition through descriptive analytics and predictive modeling.

The project combines data cleaning, exploratory data analysis, interactive Power BI dashboards, and machine learning to identify the factors influencing employee attrition and predict employees who may be at risk of leaving.

---

## Objectives

* Clean and preprocess a real-world HR dataset.
* Perform exploratory data analysis to uncover attrition trends.
* Build an interactive Power BI dashboard for HR decision-making.
* Develop a machine learning model to predict employee attrition.
* Generate business insights and recommendations to improve employee retention.

---

## Dataset

The repository contains both the raw and cleaned versions of the dataset.

### Raw Dataset

The original dataset contains various data quality issues, including:

* Missing values
* Inconsistent categorical values
* Formatting inconsistencies
* Duplicate records
* Mixed data types

### Processed Dataset

The cleaned dataset has been preprocessed and prepared for analysis and machine learning.

The preprocessing stage includes:

2. Feature engineering
3. Train/Test split
4. Model training
5. Model evaluation

Evaluation metrics include:

* Accuracy

* Recall
* F1 Score
* ROC-AUC

## Business Insights

The analysis identifies the key factors associated with employee attrition and provides actionable insights that can assist HR teams in making informed retention decisions.

The predictive model can help organizations proactively identify employees who may be at risk of leaving, enabling targeted retention strategies.
